# **App Name**: PsycheAssist

## Core Features:

- Text Input: Allows users to input text describing their interactions or situations.
- AI Analysis: Utilizes an AI model to analyze the text input for indicators of psychological abuse, acting as a tool to determine potential abusive patterns.
- Feedback & Resources: Provides users with feedback based on the AI analysis, offering insights and suggesting relevant resources.
- Safety Exit: A prominent 'Panic button' that immediately redirects the user to a neutral website in case they are being monitored.

## Style Guidelines:

- Primary color: Soft, muted blue (#A0C4FF) to evoke calmness and trust.
- Background color: Very light, desaturated blue (#F0F8FF) to provide a gentle backdrop.
- Accent color: Gentle lavender (#BDB2FF), to create a sense of safety and support.
- Body and headline font: 'Inter', a neutral sans-serif font, to provide readability and a modern feel.
- Use simple, clear icons to represent different resources and features.
- Clean and intuitive layout with clear sections for text input, analysis results, and resource suggestions.
- Subtle, gentle animations to provide feedback and guide the user through the process without overwhelming them.
